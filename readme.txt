2024/05/07 v1.0


---使い方---

・そのまま実行すればシステムタスクトレイ内に表示されます
　だだバッテリ残量なんて常に見たいのでスタートアップに登録することを推奨します

　Battery.exeを任意の場所に配置→Battery.exeのショートカットを作成
　→Win+R shell:startup→ショートカットをここに配置
　これでスタートアップに登録でき次回起動時以降自動的に実行されます

・デフォルトだと非表示のシステムタスクトレイとしてタスクバー通知欄左側の＾に隠されています。＾を押しBattery.exeが実行されているのを確認したら、
　IMEの隣あたりにD&Dしてください。そうするとタスクバー通知欄に常駐します。次回起動時以降もここに表示されるようになります。

・Battery.exeは何個でもも起動できてしまいます、そんな管理なんてしてません
・リソース開放もしてません。多分ガベージコレクションを勝手にやってくれてるはずです、詳しくは知りません。

・ソースはソースフォルダの.javaのやつです、classとかはコンパイル時の副産物(一応残してある)
　なんか変更加えたければソースを適当にいじってください

・ソフト終了時はトレイアイコンを右クリック→Closeで終了できます



---コンパイル方法---

※当たり前のようにJavaにはパスを通してください
・javacでコンパイル
　javac -encoding utf-8 BatteryTrayApp.javaを実行
　日本語使ってないはずだけどutf-8じゃないとコンパイルエラーになります。開発時の文字コードかコメントが原因？

・jarの生成
jar cvfe BatteryTrayApp.jar BatteryTrayApp BatteryTrayApp*.class
　を実行

　これですべてのクラスを含むjarファイルを生成します

・jarからexeにコンパイル
　Launch4jをインストール
　https://launch4j.sourceforge.net

　Launch4jの設定をしてexeにコンパイルするだけ
　ここらへんはググったほうがわかりやすそう

・Restart the application after a crashはオフを推奨します。何かしらソフトにエラーがありうまく起動できない時、常に再実行されるため無駄に重くなります
